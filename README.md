# Debt Management & Credit Loyalty Program and Rewards WordPress Plugin

> Plugin for Debt Management

Tired of the same old budget spreadsheets that never motivate you to pay down debt? What if each payment earned rewards and built your credit score at the same time?

This plugin transforms debt repayment from a chore into a gamified experience by letting you create a custom loyalty program that rewards every dollar paid.
